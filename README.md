ember-eui
==============================================================================

This project aims to provide ember components implementing the css layer of https://elastic.github.io/eui


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-eui
```
